2019-10-25 / 2 min read

🍒MySQL笔记第一篇 相关概念~

1.什么是数据库

1.1 概念

  • 数据库是按照数据结构来组织、存储和管理数据的仓库

1.2 常见数据库

  1. Oracle
  2. DB2
  3. SQL Server
  4. Postgre SQL
  5. My SQL

1.3 相关术语

  1. 数据库系统(Datebase System):DBS
    • 数据库
    • 数据库管理系统
    • 应用开发工具
    • 管理员及用户
  2. SQL语言
    • Structured Query Language: 结构化查询语言
      1. DDL: 数据定义语言
      2. DML:数据操作语言
      3. DQL:数据库查询语言
      4. DCL:数据控制语言
        🍋MySQL第二篇， 关于MySQL相关的操作和命令

2. 配置文件

  • my.cnf是MySQL的配置文件

3. 登录/退出 MySQL

  • 登录

    $ mysql -uroot -p
$ mysql -uroot -proot

    • 得到版本号

      $ mysql -V
$ mysql --version

    • 登录的同时打开指定数据库

      $ mysql -uroot -p -D db_name

  • 退出

    $ exit
$ quit

  • 登录信息中需要掌握的

    • 命令行结束符默认使用;或者\g来结束
    • 可以通过help或者\h或者?加上相关关键词来查看手册
    • \c可以取消当前命令的执行

4. SQL语句语法规范

  • 常用MySQL的关键字我们需要大写，库名、表名、字段名称等使用小写
  • SQL语句支持拆行操作，拆分的时候不能吧完整单词拆开
  • 数据库名称、表名称、字段名称不要使用MySQL的保留字，如果要使用，需要用反引号将其括起来

5. 常用SQL语句

SELECT USER() -- 得到登录的用户
SELECT VERSION() -- 得到 MySQL的版本信息
SELECT NOW() -- 得到当前的日期时间
SELECT DATABASE() --得到当前打开的数据库
